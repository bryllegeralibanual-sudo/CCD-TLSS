   ─────────────────────────────────────────────────────────────────────────
   Purpose:     Professional confirmation modal for important actions
   File Size:   4,506 bytes
   Features:
      • Default + Danger variants
      • Dark mode support
      • Full accessibility (ARIA, keyboard nav)
      • Smooth animations & backdrop blur
      • Design-system compliant

   Usage:
      <ConfirmDialog
        open={!!confirm}
        title="Delete Item"
        message="This cannot be undone."
        variant="danger"
        onConfirm={() => deleteItem()}
        onCancel={() => setConfirm(null)}
      />

   ─────────────────────────────────────────────────────────────────────────

   NEW COMPONENT #2: Toast.jsx
   ─────────────────────────────────────────────────────────────────────────
   Purpose:     Professional inline notifications (success/error/warn/info)
   File Size:   4,230 bytes
   Features:
      • 4 notification types with distinct icons
      • Auto-dismiss + manual close
      • Dark mode support
      • Smooth enter/exit animations
      • Configurable position & duration

   Usage:
      <Toast toast={toast} onClose={() => setToast(null)} />

      // Show notification:
      setToast({ type: 'success', message: 'Saved!' })
      setToast({ type: 'error', message: 'Error occurred' })

   ─────────────────────────────────────────────────────────────────────────
